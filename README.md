# ✒️ Calligraphy Practice Address Generator

A professional-grade utility designed to streamline calligraphy practice by generating random, realistic names and addresses. This tool replaces the need for manual brainstorming, allowing artists to focus entirely on their letterforms.

## ✨ Features

* **Dynamic Quantity:** Generate between 1 and 50 entries at a time.
* **Clean GUI:** Built with Python's `tkinter` for a minimalist, distraction-free interface.
* **Export Functionality:** Save your generated lists as `.txt` files for easy printing or digital reference.
* **Visual Clarity:** Built-in separators between entries to help maintain your place while practicing.

## 🛠️ Technical Improvements (Version 2.0)

This is a complete rebuild of my original address generator. Key upgrades include:
* **Virtual Environment:** Now managed with a dedicated `.venv` for better dependency control.
* **Git Integration:** Professional `.gitignore` setup to keep the repository clean.
* **Enhanced GUI:** Migrated to a centralized logic structure with error handling for file operations.
